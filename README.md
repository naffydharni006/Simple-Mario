# Simple Mario

In this game you will find your mario falling on hot surface and you have to use the up arrow key to take him up. Other thing is that there is a dragon in front of mario who is throwing fireballs which would kill mario. Then there are will flowers on the roof which can also kill mario when it touches them. The floor and the roof will come close to each other will your level increases. Now you have to save mario from these.
